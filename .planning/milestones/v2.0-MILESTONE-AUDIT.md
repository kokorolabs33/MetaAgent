## Cross-Phase Integration

**18/20 connections verified.** 2 semantic gaps (non-blocking):

### Gap 1: Tool Results → Artifact Cards (Semantic)

`executeWebSearch` returns human-readable prose. `detectArtifactMetadata` expects JSON with `{"artifacts": [...]}` schema. The full pipeline is wired but never fires from tool invocations because the data format doesn't match. Artifact cards will render if agents are prompted to output structured JSON or if tools return artifact-schema directly.

**Impact:** SearchResultsCard doesn't auto-populate from web search. Code blocks and markdown rendering work independently via react-markdown.

### Gap 2: Streaming Platform URL (Documentation)

`TASKHUB_PLATFORM_URL` controls where the agent POSTs streaming deltas. Default `http://localhost:8080` works locally but is undocumented in `.env.example`. Non-localhost deployments silently degrade streaming to no-op.
