  <action>
Rewrite the `Stream` handler in `internal/handlers/stream.go` to subscribe BEFORE replaying, then dedup:

1. **Subscribe first:** Move `h.Broker.Subscribe(taskID)` to BEFORE the DB replay query. The channel immediately starts buffering live events (up to 64, per broker.go buffer).

2. **Replay from DB:** Then execute the existing replay logic (ListByTask or ListByTaskAfter depending on Last-Event-ID header). Write replayed events to the SSE stream. Collect replayed event IDs into a `map[string]bool` set for dedup.

3. **Drain buffered + stream live:** After replay completes, enter the existing select loop on `ch` and `ctx.Done()`. Before writing each live event from the channel, check if its `evt.ID` is in the dedup set. If yes, skip it (it was already sent during replay). If no, write it. After the dedup set has been used for ~5 seconds of streaming (or after the first 100 live events), clear it to free memory — any event arriving that late cannot be a duplicate of replay.

4. **Keep the defer unsubscribe:** `defer h.Broker.Unsubscribe(taskID, ch)` should remain right after Subscribe.

The key ordering change: line 73 (`ch := h.Broker.Subscribe(taskID)`) must move to before line 41 (the replay query block).

Concrete structure:
```go
// 1. Subscribe to live events FIRST (per D-01)
ch := h.Broker.Subscribe(taskID)
defer h.Broker.Unsubscribe(taskID, ch)

// 2. Replay historical events
var replayEvents []models.Event
// ... existing Last-Event-ID logic ...

// 3. Write replayed events, track IDs for dedup (per D-02)
seen := make(map[string]struct{}, len(replayEvents))
for i := range replayEvents {
    writeSSEEvent(w, &replayEvents[i])
    seen[replayEvents[i].ID] = struct{}{}
}
flusher.Flush()

// 4. Stream live events with dedup
for {
    select {
    case <-ctx.Done():
        return
    case evt, ok := <-ch:
        if !ok {
            return
        }
        if _, dup := seen[evt.ID]; dup {
            delete(seen, evt.ID) // Remove after first hit
            continue
        }
        writeSSEEvent(w, evt)
        flusher.Flush()
    }
}
```

Do NOT change writeSSEEvent, the Broker, or the Store. Only the Stream handler function changes.

```go
import (
    "github.com/anthropics/anthropic-sdk-go"
    "github.com/anthropics/anthropic-sdk-go/option"
)

func callLLM(ctx context.Context, apiKey, systemPrompt, userMsg string) (string, error) {
    if apiKey == "" {
        return "", fmt.Errorf("ANTHROPIC_API_KEY is not set — task execution requires an Anthropic API key")
    }

    client := anthropic.NewClient(option.WithAPIKey(apiKey))

    message, err := client.Messages.New(ctx, anthropic.MessageNewParams{
        Model:     anthropic.ModelClaude3_5Sonnet20241022,
        MaxTokens: 4096,
        System: []anthropic.TextBlockParam{
            anthropic.NewTextBlock(systemPrompt),
        },
        Messages: []anthropic.MessageParam{
            anthropic.NewUserMessage(anthropic.NewTextBlock(userMsg)),
        },
    })
    if err != nil {
        return "", fmt.Errorf("anthropic api: %w", err)
    }

    // Extract text content from response
    var result string
    for _, block := range message.Content {
        if block.Type == anthropic.ContentBlockTypeText {
            result += block.Text
        }
    }

    result = stripMarkdownFences(strings.TrimSpace(result))
    return result, nil
}
```
