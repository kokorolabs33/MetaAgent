# Agent Skills

Skills provide structured workflows for common operations. Reference them in conversations
using `$skill-name` syntax.

## Available Skills

| Skill | Path | When to Use |
|-------|------|-------------|
| `$code-review` | `.agents/skills/code-review/SKILL.md` | Reviewing code changes or PRs |
| `$pr-maintainer` | `.agents/skills/pr-maintainer/SKILL.md` | Triaging, reviewing, or landing PRs |
| `$add-feature` | `.agents/skills/add-feature/SKILL.md` | Adding new functionality (full-stack checklist) |
